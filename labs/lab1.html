Лабораторная работа 1: Интерфейс Unity

**Цель работы**: Изучить основные элементы интерфейса Unity.

**Материалы и оборудование**: Компьютер с установленным Unity.

**Описание**:

Unity — это мощный инструмент для разработки игр, который предоставляет множество возможностей для создания интерактивных 2D и 3D приложений. В этой лабораторной работе мы рассмотрим основные элементы интерфейса Unity.

**1. Главный экран Unity**:

Главный экран Unity состоит из нескольких областей:

  • Scene — область, где создается и редактируется сцена.
  • Game — область, где отображается результат выполнения вашей игры.
  • Hierarchy — область, содержащая все объекты сцены.
  • Project — область, где хранятся все ресурсы проекта.
  • Inspector — область, где можно настроить свойства выбранных объектов.

**2. Область Scene**:

Область Scene предназначена для работы с объектами в трехмерном пространстве. Здесь вы можете перемещать, масштабировать и вращать объекты.

**3. Область Game**:

Область Game используется для просмотра игры так, как она будет выглядеть во время выполнения.

**Задание**: Изучите основные элементы интерфейса Unity, следуя описанию и картинкам выше. Перемещайтесь между различными областями, чтобы понять их функции.

**Ожидаемые результаты**: Учащиеся должны получить базовое понимание интерфейса Unity и уметь ориентироваться в его основных компонентах.
